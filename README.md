readme = """# Crechè Preschool

**A colorful, playful, privacy-respecting early-learning portal for young children and their grown-ups.**

Crechè Preschool is the early-learning gateway within the Verve N Veda and Khaemenes Academy ecosystem. It brings alphabet activities, beginning reading, numbers, shapes, art, music, movement, emotions, weather, and playful discovery into one welcoming browser-based space.

**Live site:** `https://vervenveda.github.io/preschool.github.io/`

---

## Welcome to Crechè

Crechè is designed to feel like a cheerful preschool lobby rather than a traditional directory. Children can enter a bright activity garden, choose a learning room, save favorite activities, or let the page select a surprise adventure.

The current landing page uses flowing blue, aqua, cream, coral, orange, peach, and teal shapes inspired by child-friendly classroom art. The interface avoids visual clutter while keeping the experience energetic, inviting, and easy to navigate.

---

## Project Goals

Crechè Preschool is being developed to provide:

- Free early-learning activities in an accessible web browser
- Large, colorful controls suitable for young learners
- Simple navigation for children, parents, caregivers, and educators
- A safe, calm alternative to advertising-heavy children’s websites
- Portable, independent HTML applications
- Local browser favorites without requiring an account
- A growing bridge into Khaemenes Academy and the wider Verve N Veda network

---

## Current Features

The landing page includes:

- A colorful abstract preschool welcome area
- Learning activities organized by subject
- Search by title, topic, description, or filename
- Category filters
- A rotating daily learning adventure
- A **Surprise Me** activity selector
- Favorite activities stored locally in the browser
- A favorites-only viewing option
- Copy-link controls for parents and educators
- A calmer visual mode
- A searchable slide-out navigation menu
- A clearly separated grown-up section
- Responsive layouts for desktop, tablet, and mobile
- Keyboard navigation and visible focus states
- Reduced-motion support
- Print-friendly formatting
- No analytics, advertising, tracking scripts, or required accounts
